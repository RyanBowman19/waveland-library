# Putting this site into WordPress

**Use the easy, ready-to-paste version instead of this file.**

Everything you need is in the **`wordpress-ready/`** folder. Open
**`wordpress-ready/READ-ME-FIRST.md`** and follow it — the files there are pre-built to
copy and paste straight into WordPress, with the styling already embedded and the photos
uploaded into the library's own Media Library. Nothing depends on any outside account, so
it's fully the library's to own.

(This file is kept only as a pointer; the older step-by-step write-up was replaced by the
simpler `wordpress-ready/` approach.)
